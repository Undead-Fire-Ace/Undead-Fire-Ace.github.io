一篇题解

# 题目传送门

# 简要题意：

问这给定的 nnn 对舞伴中只有 kkk 对女伴比男伴高的方案数

# 分析：

先浅浅的看一下题干，要求组合数但是没有给 modmodmod 而且 NNN 和 KKK 达到了 200，这必然会爆掉，然后就可以得出一个显然易见结论，要写高精度（不想写开溜）。

先浅挂一下高精度的板子，因为有些人可能就懒得写了（doge。下面挂的是重载运算符的板子，可以省去函数名的重复敲击

然后又想，高精度是个 (n2)(n^2)(n2) 复杂度的算法，这就必然不能用 (n3)(n^3)(n3) 的状态转移方程了。因此我们先对男生和女生分别排序，どうしてですか？因为我们发现排序后会有单调性就更有利于我们的状态转移，然后设 dp(i,j)dp(i,j)dp(i,j) 为前 iii 个男生和女生已经两两匹配总共有 jjj 对符合题意的方案数。那么最终的答案就应该是 ∑i=0kdp(n,i)\sum_{i = 0}^{k}dp(n,i)∑i=0k​dp(n,i) 。

接下来就要考虑怎么转移状态了，假如当前的女生的身高小于等于男生的身高：

# 当前女生不高于男生:

# ①：这个女生不使 jjj 增加。

然后将这个女生放在身高大于等于她的那个男生的位置，就会形成一个新的不会影响jjj 的数量的方案。

我们可以从当前位置向前扫一遍就可以得到符合 这种情况的位置的个数，暂且设个数为 kkk 。我们也可以将当前女生放到任意原来就是女生更高的位置。由于女生身高已经过排序，交换后 “原来就是女生更高的位置” 仍然是女生更高，而当前位置也仍然是男生不矮于女生。

因为我们枚举的时候是从小到大枚举的，更换位置是将更高的放在前面，而这样的女生在换过来之后就一定是小于当前的男生的，显而易见，这样的位置有 jjj 个。

并且 kkk 和 jjj 不会重叠，因为kkk 位置的部分满足男生比女生高。所以，对于 dp(i,j)dp(i,j)dp(i,j) 共有 j+kj+kj+k 个位置使 jjj 不增加

# ②: 这个女生使 jjj 增加.

因为刚才已经求出了不会使 jjj 增加的位置数，那么用总数 iii 减去上面的个数就是能使 jjj 增加的个数

然后这种情况下的递推方程为:

dp[i][j]=dp[i−1][j]×(j+k)+dp[i−1][j−1]×(i−k−j−1)dp[i][j]\ = \ dp[i - 1][j] \ \times \ (j \ + \ k ) \ + \ dp[i \ - \ 1][j \ - \ 1]\ \times \ (i-k-j-1)dp[i][j] = dp[i−1][j] × (j + k) + dp[i − 1][j − 1] × (i−k−j−1)

第一类的情况解决掉了，接下来看第二类情况。

# 当前的女生比男生高：

# ①：当前女生不使 jjj 增加。

这个时候无论将她放在哪里都会形成一组女生比男生高的组合，因为此时的女生一定高于当前所枚举的所有男生。我们现在考虑她不影响 jjj 的位置有哪些，然后可以轻松地发现其实将她放在一个之前就是女生比男生高的位置同时那个位置的女生不高于当前位置的男生。

女生高于男生的位置数就是已经求出来的 jjj ，然后再扫一遍得出有多少女生高于当前男生，计个数为 kkk 个，那么满足不影响 jjj 的位置数就是有 j−kj - kj−k 个。

# ②：当前女生使 jjj 增加

同理第一大类的这种情况，总数 iii 减去上面的方案数就行。

那么我们也就得出了这类情况的转移方程：

dp[i][j]=dp[i−1][j]×(j−k)+dp[i−1][j−1]×(i−j+k+1)dp[i][j] \ = \ dp[i - 1][j] \ \times\ (j-k) \ + \ dp[i-1][j-1] \ \times \ (i-j+k+1)dp[i][j] = dp[i−1][j] × (j−k) + dp[i−1][j−1] × (i−j+k+1)

最后的最后，因为我们发现转移过程只与 iii 和 i−1i - 1i−1 有关系，因此可以用滚动数组来优化空间，（其实这道题的数据规模不用滚动数组优化空间也行，而且会跑到更快，但是还是要学一下这种思想的）具体代码实现方式会在下面的代码中说明
