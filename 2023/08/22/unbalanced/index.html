# [AGC045B] 01 Unbalanced

# 题意：

给定一个字符串，其中 ??? 可以替换成 000 或 111。让最小化 000 和 111 的数量差。

# 分析：

一般对于最小化最大值的题，往往都可以用二分去做，（但是这道题的二分的 check 函数我不会写），那么就想一想有没有什么平替，我们可以想到贪心和 dp 也是可以解决类似的问题的，我们可以考虑因为是跟个数有关且只有两个不同的数，那么我们就可以将 000 看成是 −1-1−1，111 是 +1+1+1，然后就转换成了最小化最大的累积和与最小的累积和的差值。先可以设，在累计和最大值不超过 limitlimitlimit 时，flimitf_{limit}flimit​ 为最大累计和的最小值，ggg 为最小累计和的最大值，那么我们的答案就是求 min⁡{limit−flimit}(g≤limit)\min \left\{limit - f_{limit}\right\}(g\leq limit)min{limit−flimit​}(g≤limit)。

然后我们需要去求 ggg，对于求 ggg，我们可以贪心的去求解，在一开始我们先将所以的问号都设置成 −1-1−1，然后在检查一遍，如果累计和不超过我们的 limitlimitlimit 的话，就将 −1-1−1 变成 111。

那么这样贪心为什么是正确的呢？

当 limitlimitlimit 增加 222 时，最多只有一个 ??? 发生变换，所以 flimit+2≤flimit+2f_{limit+2}\leq f_{limit}+2flimit+2​≤flimit​+2，那么 limit−flimit+2≤(limit+2)−(flimit+2)limit-f_{limit+2}\leq (limit+2)-(f_{limit}+2)limit−flimit+2​≤(limit+2)−(flimit​+2)，那么我们最终的答案为 min⁡{limit−flimit}≤min⁡{(limit+2)−flimit+2}\min \left\{limit-f_{limit} \right\} \leq \min \left\{ (limit+2)-f_{limit+2} \right\}min{limit−flimit​}≤min{(limit+2)−flimit+2​}。

那么正确性保证了，最后的答案也就是 min⁡(g−fg,(g+1)−fg+1)\min(g-f_{g},(g+1)-f_{g+1})min(g−fg​,(g+1)−fg+1​)。

代码实现起来也是比较容易的，一些细节也在代码中标注了。
